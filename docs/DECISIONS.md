# Decisões Técnicas

## Regra #8

Nunca automatizar a interface gráfica se o HTML já fornecer os dados.

## Regra #9

Não recalcular o que o SIGAA já calcula corretamente.

## Regra #10

Sempre extrair informações seguindo a hierarquia natural do SIGAA.

## Regra #11

Uma disciplina pode ter múltiplas fontes de atividades.

## Regra #12

Uma entidade nunca deve duplicar informações de outra entidade.

# Decisão 3

Evento deixa de existir como entidade independente.

Todo evento será modelado como um Compromisso.

Motivo:

Evitar duplicação de responsabilidades.
