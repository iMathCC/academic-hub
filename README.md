# 🎓 Academic Hub

Painel acadêmico pessoal para centralizar informações do SIGAA, Google Classroom e Google Calendar em uma interface minimalista e responsiva.

## Objetivo

Eliminar a necessidade de abrir o SIGAA para tarefas simples, concentrando as informações importantes em um único ambiente.

## Tecnologias

- Python
- Flask
- SQLite
- HTML
- CSS
- Bootstrap
